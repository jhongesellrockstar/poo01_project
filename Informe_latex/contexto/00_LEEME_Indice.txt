PATITAS SEGURAS — Documentación modular (formato .txt)
Fecha: 2025-12-18

Este paquete contiene una explicación técnica y modular del proyecto de software “Patitas Seguras”, una aplicación de escritorio orientada a la adopción, búsqueda de mascotas extraviadas y mecanismos comunitarios de prevención del maltrato mediante listas negras y ranking de veterinarias.

NOTA
- El enfoque es modular: cada archivo describe un componente, su responsabilidad, su interacción con otros módulos y criterios de implementación.
- Se asume una base tecnológica Python + Tkinter, mapas con TkinterMapView, tratamiento de imágenes con Pillow y persistencia con SQLite.
